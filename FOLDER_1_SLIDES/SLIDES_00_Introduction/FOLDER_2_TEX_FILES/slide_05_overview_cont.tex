%%%%%%%%%%%%%%%%%
% Itemize begin %
%%%%%%%%%%%%%%%%%
\begin{itemize}

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
% Intermediate Representation %
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\item Instead of direct \fbox{src $\rightarrow$ dst} translation, do \fbox{src $\rightarrow$ IR $\rightarrow$ dst}
\item IR contains everything needed for translation to machine code
\item IR is effective for handling \textit{multiple src and dst languages}
\item Different compilers have different IRs
\item IR features ideally \textit{independent of src language}
\item IR features ideally \textit{independent of dst language}
\item Designing a good IR is more art than science

%%%%%%%%%%%%%%%
% Itemize end %
%%%%%%%%%%%%%%%
\end{itemize}
